# Operators 

In order to have a boolean results (true,false) you need to use comparison operators, logical comparison: 


### comparison operators:

1. == is equal to. 
2. != is not equal to.
3. === strict equal to. 
4. !== strict Not equal to. 

***note***: strict comparison dose compare value and data type. 

### logial operators:
1. && logical AND. 
2. \||\ logical OR.
3. ! logical NOT. 


# loops:
